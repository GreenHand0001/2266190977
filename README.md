## 前言

在大学校园中，二手书籍的买卖一直是一个活跃的市场。为了方便广大师生进行二手书籍的交易，我们开发了这个基于Spring Boot的校园二手书籍拍卖小程序。以下是关于该项目的详细介绍。

## 内容介绍

本项目主要实现了一个线上二手书籍拍卖平台，用户可以在平台上发布自己要出售的书籍，也可以浏览、搜索并竞拍其他用户的书籍。平台提供了完整的拍卖流程，包括出价、竞拍、支付和评价等环节。此外，我们还提供了丰富的功能，如分类浏览、搜索、书籍推荐等，以提升用户体验。

## 技术介绍

- 语言：Java
- 使用框架：Spring、Spring MVC，MyBatis，微信小程序
- 前端技术：JS、Vue、CSS3，Uniapp
- 开发工具：IDEA/Eclipse，Uniapp
- 数据库：MySQL 5.7/8.0
- 数据库管理工具：phpstudy/Navicat
- JDK版本：jdk1.8
- Maven：apache-maven 3.8.1-bin
- 前端环境：Node.Js 12\14\16
